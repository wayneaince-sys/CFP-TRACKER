# Big Sarge College Football Championship Playoffs Tracker

An interactive polling page for the College Football Playoff matchup between Miami Hurricanes and Indiana Hoosiers on January 19, 2025.

## Features

- **Win Probability Analysis**: Data-driven predictions based on season stats, injuries, and key factors
- **Interactive Voting**: Users can vote for their predicted winner
- **Real-Time Results**: Animated bar chart showing live poll results
- **Responsive Design**: Works on desktop, tablet, and mobile devices
- **Vote Persistence**: Uses localStorage to remember votes and prevent duplicate voting

## Win Probability Breakdown

### Miami Hurricanes (58%)

- Strong offensive efficiency: 34.2 PPG
- Elite QB play with Cam Ward
- Moderate playoff experience
- Playing on neutral site

### Indiana Hoosiers (42%)

- Outstanding defense: 15.8 PPG allowed
- Impressive 11-1 record
- High motivation as underdog
- Less playoff experience

- **Real-time Updates** - Polls localStorage every 3 seconds
